code Git & GitHub CLI

One-page reference for Git and gh — searchable, forkable, always free.
search
Repository Management setup
git init Initialize a new Git repository
git clone <url> Clone a remote repository
git clone --depth 1 <url> Shallow clone (faster, no history)
git remote add origin <url> Add a remote named 'origin'
git remote -v List remotes (verbose)
git remote set-url origin <url> Change remote URL
git remote rm <name> Remove a remote
Basic Workflow workflow
git status Show working tree status
git status -s Short status (one line per file)
git add <file> Stage a file
git add . Stage all changes (current dir)
git add -A Stage all changes (whole repo)
git add -p Interactive staging (hunk-by-hunk)
git commit -m "msg" Commit staged changes with message
git commit -am "msg" Stage & commit tracked files
git diff Show unstaged changes
git diff --staged Show staged changes
git log Show commit log
git log --oneline Compact log (one line per commit)
git log --oneline --graph --all Graphical log of all branches
git log -p Log with diffs
Branch & Merge branch
git branch List local branches
git branch -a List all branches (local + remote)
git branch <name> Create a new branch
git branch -d <name> Delete a branch (safe)
git branch -D <name> Force-delete a branch
git branch -m <old> <new> Rename a branch
git checkout <branch> Switch to a branch
git checkout -b <name> Create & switch to new branch
git switch <branch> Switch to a branch (modern)
git switch -c <name> Create & switch (modern)
git merge <branch> Merge branch into current
git merge --no-ff <branch> Merge with no fast-forward
git rebase <branch> Rebase current onto branch
git rebase -i HEAD~n Interactive rebase last n commits
Remotes & Collaboration remote
git push Push to default remote
git push origin <branch> Push branch to origin
git push -u origin <branch> Push & set upstream
git push --force Force push (⚠ overwrites)
git push --force-with-lease Safer force push (checks remote)
git push -d origin <branch> Delete remote branch
git pull Fetch & merge (default remote)
git pull --rebase Fetch & rebase instead of merge
git fetch Fetch from remote (no merge)
git fetch --prune Fetch & remove stale refs
git fetch --all Fetch from all remotes
git cherry-pick <commit> Apply a specific commit to current
Undo & History undo
git restore <file> Discard unstaged changes (modern)
git restore --staged <file> Unstage a file (modern)
git reset HEAD <file> Unstage a file (classic)
git reset --soft HEAD~1 Undo last commit, keep changes staged
git reset --mixed HEAD~1 Undo last commit, unstage changes
git reset --hard HEAD~1 Undo last commit, discard changes (⚠ destructive)
git revert <commit> Create a new commit undoing the given commit (safe for shared history)
git commit --amend Edit last commit message or include staged changes
git blame <file> Show who changed each line with commit info
GitHub CLI gh
gh auth login Authenticate gh with GitHub
gh repo create <name> Create a new repo on GitHub
gh repo clone <owner>/<repo> Clone a repo by name
gh pr create Create a pull request
gh pr list List open PRs
gh pr checkout <num> Checkout a PR locally
gh pr view View PR details in terminal
gh issue list List open issues
gh issue create Create a new issue
gh workflow run <name> Trigger a workflow by name
gh run list List recent workflow runs
gh run watch <id> Watch a workflow run live
gh run download <id> Download workflow run artifacts
Stash & Clean stash
git stash Stash working directory changes
git stash pop Apply & remove latest stash
git stash list List all stashes
git stash apply stash@{n} Apply a specific stash (keep it)
git stash drop stash@{n} Delete a specific stash
git stash clear Delete all stashes
git clean -n Dry-run: show untracked files to remove
git clean -fd Remove untracked files & directories
Config & Aliases config
git config --global user.name "X" Set global user name
git config --global user.email "x@y" Set global user email
git config --global alias.<name> "cmd" Create a global alias
git config --global core.editor "nvim" Set default editor
git config --global init.defaultBranch main Default branch name to 'main'
git config --list List all config
git config --global -e Open global config in editor
git config --global push.autoSetupRemote true Auto-set upstream on push (git push without -u)
Advanced Git advanced
git bisect start Start binary search for a bug
git grep "pattern" Search working tree for pattern
git tag List tags
git tag -a v1.0 -m "msg" Create annotated tag
git worktree add <path> <branch> Create a linked worktree
git submodule add <url> <path> Add a submodule
git submodule update --init --recursive Clone & init all submodules
git config --global rerere.enabled true Reuse recorded resolution of merge conflicts
git reflog Show reference log (recover lost work)

Cursor Movement / Motions motions
h j k l ← ↓ ↑ →
w / b Next / previous word start
e / ge Next / previous word end
W / B Next / previous WORD (non-blank)
0 Go to line beginning
$ Go to line end
^ Go to first non-whitespace of line
_ Go to first non-whitespace of line (count-1 lines down)
gg Go to first line
G Go to last line
H / M / L Top / Middle / Bottom of screen
% Jump to matching bracket
fx Forward to next occurrence of x on line
Fx Backward to previous occurrence of x
tx Forward until just before x
Tx Backward until just after x
; / , Repeat last f/t in same / opposite direction
} / { Next / previous empty line (paragraph jump)
ctrl-d / ctrl-u Scroll down / up half page
ctrl-f / ctrl-b Page down / up
zz / zt / zb Center / Top / Bottom of cursor on screen
Editing — Insert / Change / Delete / Yank / Paste editing
i / a Insert before / after cursor
I / A Insert at line start / end
o / O Open new line below / above
s / S Delete char / line and enter Insert mode
c + motion Change (delete + insert) — e.g. cw, ciw
cc / C Change whole line / change to line end
d + motion Delete — e.g. dw, diw, d$
dd / D Delete whole line / delete to line end
x / X Delete (cut) char under / before cursor
y + motion Yank (copy) — e.g. yw, yiw
yy / Y Yank whole line / yank to end of line
p / P Paste after / before cursor
gp / gP Paste and leave cursor after pasted text
r Replace single character
R Enter Replace (overwrite) mode
. Repeat last change
u / ctrl-r Undo / Redo
g; / g, Jump to older / newer change position
Visual Mode visual
v Enter Visual (character-wise)
V Enter Visual (line-wise)
ctrl-v Enter Visual (block-wise)

aw / iw Select a / inner word

a{ / i{ Select around / inside curly braces
a[ / i[ Select around / inside brackets
a< / i< Select around / inside angle brackets
a" / i" Select around / inside double quotes
a' / i' Select around / inside single quotes
a` / i` Select around / inside backticks
at / it Select around / inside HTML tag
gv Reselect last selection
> / < Indent / Dedent selection
~ / u / U Toggle / lowercase / UPPERCASE
Search & Replace search
/pattern Search forward for pattern
?pattern Search backward for pattern
n / N Next / previous match
* / # Search forward / backward for word under cursor

:%s/old/new/gc Replace with confirmation per match
:%s/old/new/gi Case-insensitive replace all
:'<,'>s/old/new/g Replace in visual selection
g& Repeat last :s across all lines
nohlsearch Remove search highlighting (can also :noh)

Terminal multiplexer — sessions, windows, panes & more.

Session Management session
tmux new -s <name> Create a new named session
tmux new -s <name> -d Create session detached (background)
tmux ls List all sessions
tmux a Attach to last session
tmux a -t <name> Attach to named session
tmux kill-session -t <name> Kill a session
tmux kill-session -a Kill all sessions but current
tmux switch -t <name> Switch to another session
tmux rename-session -t <old> <new> Rename a session
Window Management window
Prefix + c Create a new window
Prefix + , Rename current window
Prefix + & Kill current window
Prefix + w List / navigate windows
Prefix + p / Prefix + n Previous / next window
Prefix + <number> Go to window by number
Prefix + l Go to last active window
Prefix + f Find window by name
Prefix + . Move window (change index)
Prefix + , Set window title
Pane Management pane
Prefix + % Split pane vertically
Prefix + " Split pane horizontally
Prefix + o Cycle through panes
Prefix + ; Go to last active pane
Prefix + x Kill current pane
Prefix + q Show pane numbers
Prefix + z Toggle full-screen zoom for pane
Prefix + ! Convert pane into its own window
Prefix + { / Prefix + } Swap pane left / right (up / down)
Prefix + ctrl-o Rotate pane positions forward
Prefix + Space Cycle through pane layouts
Prefix + M-1 .. M-5 Select preset layout (even-h, even-v, main-h, main-v, tiled)
Prefix + M-Up/Down/Left/Right Resize pane by 5 cells
Prefix + ctrl-Up/Down/Left/Right Resize pane by 1 cell
Copy Mode & Scrolling copy

{vim-like Vim motions work in copy mode (if vi mode enabled)

Prefix + = Show paste buffer history and choose
